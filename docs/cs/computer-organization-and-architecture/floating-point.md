# 浮点数

- 通常使用 16/32 个 bit 来表示一个数，但是使用定点数来表示会使得原来可以映射 40 亿实数变成只能映射 1 一个实数
- 浮点数解决了这个问题

在计算机理，使用科学计数法用来表示比较大的数。

IEEE 标准定义了两个基本的格式。32 bit 表示单精度的浮点数，即常说的 float 或 float 32。另一个是用 64 bit 表示的双精度的浮点数，即 double 或者 float64。

单精度可以分为三部分：

1. 符号位，用来表示正数还是负数
2. 指数位
3. 有效数位
