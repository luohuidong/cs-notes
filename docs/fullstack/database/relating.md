## 子查询

子查询（Subquery）是嵌套在另一个 SQL 查询中的查询，因此也称为嵌套查询（Nested Query）。子查询可以用于 SELECT、INSERT、UPDATE 或 DELETE 语句中，以提供动态生成的结果集。
